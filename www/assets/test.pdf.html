1. Server Overview

This document describes the multithreaded web server implementation for the Operating Systems 2025 course project.

The server is designed to handle concurrent HTTP/1.1 requests using a combination of process pooling and thread pooling techniques.

// Server Configuration Example ServerConfig config = { port: 8080, max_threads: 16, max_processes: 8, cache_size: 100, timeout: 30, log_level: "INFO" };

3. Supported File Types

The server correctly handles the following MIME types:

  • HTML documents (.html, .htm)
  • CSS stylesheets (.css)
  • JavaScript files (.js)
  • JSON data (.json)
  • Images (.png, .jpg, .svg, .gif)
  • PDF documents (.pdf)
  • Plain text (.txt)
4. Error Handling

The server implements comprehensive error handling for the following HTTP status codes:

  • 400 - Bad Request
  • 404 - Not Found
  • 408 - Request Timeout
  • 500 - Internal Server Error
  • 501 - Not Implemented
  • 503 - Service Unavailable

Each error page provides relevant information and suggestions for resolution.
